# ЗАВДАННЯ: Пошук клієнтів для платформи BetonKo

## Про платформу

**BetonKo** — B2B маркетплейс для купівлі-продажу бетону в Україні. Платформа з'єднує покупців бетону (будівельні компанії, приватних забудовників) з виробниками (бетонні заводи). Покупець створює замовлення з потрібними параметрами — виробники надсилають зустрічні пропозиції з цінами. Покупець обирає найкращу пропозицію.

## ДОДАТОК: Повний перелік технічних характеристик бетону

### Марки бетону (за міцністю):
- **М100** — 98.23 кг/см² — підготовчі роботи
- **М150** — 131.0 кг/см² — легкі фундаменти, підлоги
- **М200** — 196.0 кг/см² — фундаменти, стіни (найпопулярніша)
- **М250** — 261.9 кг/см² — монолітні конструкції
- **М350** — 327.4 кг/см² — перекриття, колони, басейни
- **М400** — 392.9 кг/см² — мости, гідроспоруди
- **М450** — 458.4 кг/см² — метро, тунелі
- **М500** — 523.7 кг/см² — спеціальні конструкції
- **М700** — високоміцний бетон
- **М800** — надміцний бетон

### Класи бетону (гарантована міцність, МПа):
В7.5, B12.5, B15, B20, B25, B30, B35, B40, B50, B55

### Морозостійкість (кількість циклів):
F50, F100, F150, F200, F300, F400, F600, F800, F1000

### Водостійкість (тиск води, атм):
W2, W4, W6, W8, W10

### Рухливість суміші (осад конусу):
- П1 (S1) — 1-5 см — жорстка
- П2 (S2) — 5-10 см — малорухлива
- П3 (S3) — 10-15 см — рухлива (найпоширеніша)
- П4 (S4) — 15-20 см — литий бетон
- П5 (S5) — >20 см — текучий

### Зимові добавки:
- Без зимових добавок — для роботи при +5°C і вище
- Гаряча вода — для температур до -5°C
- Добавка М-5 — для температур до -5°C
- Добавка М-10 — для температур до -10°C
- Добавка М-15 — для температур до -15°C
